1)Download the required XAMPP based on your OS
https://www.apachefriends.org/download.html (Recommended version PHP 7.4.3)

2)Download ATOM IDE to execute the PHP software
https://atom.io/

3)Open Xampp and Start Apache and MySQL servers

4)Extract the zip file inside the disk LocalDisk/xampps/htdocs/ where you installed and then delete the zip

4)Open Atom IDE, import the LocalDisk/xampps/htdocs/icare from import tab

5)Go back to Xampp and nearby MySQL click Admin, it should open our tab your default browser

6)To launch icare login page navigate to http://localhost/icaresystem/login.php
